MALTAS

Patagonia

PATAGONIA es la marca de las maltas para cervecería de maltexco (Ex-malterías unidas). Empresa Chilena que produce una amplia variedad de maltas para cervecerías industriales y artesanales.

Pilsen

Malta base para hacer todos los estilos de cerveza especialmente estilos alemanes y checos.

Pale Pale

Malta base para hacer todos los estilos de cerveza especialmente estilos americanos e ingleses.

Vienna

Malta base para hacer todos los estilos de cerveza. Usada especialmente como soporte para incrementar el sabor y aroma a malta.
